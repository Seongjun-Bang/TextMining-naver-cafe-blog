# 📝 Naver Blog & Cafe Sentiment Analysis

## 📌 프로젝트 개요
본 프로젝트는 현대/기아 인증중고차에 대한 소비자 인식을 분석하기 위해 네이버 블로그와 카페 데이터를 크롤링하고, 감성 분석을 통해 긍정/부정 반응을 정량적으로 파악하는 것을 목표로 합니다.

---

## 🎯 문제 정의
- 대기업(현대/기아)의 중고차 시장 진출 이후 소비자 인식은 어떠한가?
- 텍스트 데이터를 기반으로 사용자 반응을 정량적으로 분석할 수 있는가?

---

## 📊 데이터 수집
- 데이터 출처: 네이버 블로그, 카페
- 키워드: "현대기아 인증중고차"
- 수집 기간: 2024.06.01 ~ 2025.05.31
- 데이터 수: 총 420개 (블로그 210개 + 카페 210개)

---

## 🛠 데이터 전처리
- 불용어 제거 (Counter 기반 빈도 분석 활용)
- 의미 없는 단어 제거 및 데이터 정제
- 과도한 제거로 인한 데이터 왜곡 방지 위해 필터링 기준 조정

---

## 🤖 분석 방법

### 1. 텍스트 데이터 마이닝
- 워드클라우드 생성
- 주요 키워드 분석
  - "현대/기아", "서비스", "가격", "매매", "진단" 등 주요 단어 확인

### 2. K-Means 클러스터링
- Elbow Method 기반 k=4 선정
- 클러스터별 주제 분석
  - Cluster 0: 사용자 질문 및 정보 탐색
  - Cluster 1: 제도 및 시스템 설명
  - Cluster 2: 가격 및 품질 관련 정보
  - Cluster 3: 서비스 경험 및 만족도

### 3. 감성 분석
- KNU 감성사전
- EmoLex 감성사전

---

## 📈 결과

### ✔ KNU 감성사전 결과
- 긍정 비율 55% 이상
- 중립 포함 시 약 90% 수준

### ✔ EmoLex 감성사전 결과
- 감정 요소를 0/1 기반으로 분석
- 좋아요 수, 댓글 수 등과 감정 간 상관관계 분석

---

## 💡 인사이트
- 현대/기아 인증중고차에 대한 소비자 반응은 전반적으로 긍정적
- 대기업 진입이 신뢰도 상승에 영향을 미친 것으로 해석 가능
- 텍스트 데이터 분석을 통해 소비자 인식을 정량적으로 파악 가능

---

## 🚀 개선 방향
- 자동차 커뮤니티 등 다양한 데이터 소스 추가
- 딥러닝 기반 모델(BERT 등) 적용
- 감성 분석 정밀도 향상

---

## 🧠 느낀 점
단순 텍스트 분석을 넘어 실제 사회적 이슈(중고차 시장 신뢰 문제)를 데이터로 분석할 수 있었으며, 데이터 수집 범위에 따라 결과가 달라질 수 있음을 체감했습니다.
